Displaying Code - Block vs Inline Elements

Code Example 1: Block Elements vs Inline Elements Layout

<div class="block">
    This is a block element (<div>). It starts on a new line and takes up the full width.
</div>

<span class="inline">This is an inline element (<span>).</span>
<span class="inline">It stays inline with the other content.</span>

Code Example 2: Containing Other Elements

<div class="block">
    <h3>This is a heading inside a block element.</h3>
    <p>This is a paragraph inside the block element.</p>
</div>

<a href="#">This is an inline element (<a>) inside text.</a>

Code Example 3: Full-width vs Content Width

<div class="block">
    This block element takes the full width of the parent container.
</div>

<span class="inline">This inline element only takes up as much width as it needs.</span>

Code Example 4: Inline Elements Inside Block Elements

<p>
    This is a block element that contains inline elements like <strong>bold text</strong> 
    and <em>italic text</em>.
</p>
<p>
    Here is another inline element: a <a href="#">link</a> inside a paragraph.
</p>
