CSS Display and Positioning

The display property specifies the display behavior of an element. It determines how an element is rendered on the page, influencing layout and positioning.

The positioning properties define the positioning of elements in relation to their parent elements or the viewport. This allows for flexible layouts and overlapping content.

Display Property Values

  • block: The element takes up the full width available and starts on a new line.
  • inline: The element only takes up as much width as necessary, and does not start on a new line.
  • inline-block: The element flows like an inline element but allows setting width and height.
  • flex: Enables a flexible layout for child elements.
  • grid: Creates a grid layout for child elements.

Display Property Example

HTML and CSS Code

<div class="box"></div>
<div class="box"></div>

.box {
    width: 200px;
    height: 100px;
    background-color: #4CAF50;
    margin: 10px;
    display: inline-block; /* Change this value to block, flex, grid, etc. */
}

Positioning Property Values

  • static: The default value; elements are positioned according to the normal flow of the document.
  • relative: Positions the element relative to its normal position.
  • absolute: Positions the element relative to its nearest positioned ancestor.
  • fixed: Positions the element relative to the viewport, remaining in place when scrolled.
  • sticky: Toggles between relative and fixed, depending on the scroll position.

Positioning Example

HTML and CSS Code

<div class="position-example">
    <div class="absolute-box"></div>
</div>

.position-example {
    position: relative; /* Establishes a positioning context */
    background-color: #f9c74f;
    width: 150px;
    height: 150px;
    margin-top: 20px;
}
.absolute-box {
    position: absolute; /* Positions this box relative to its parent */
    top: 20px;
    left: 20px;
    width: 50px;
    height: 50px;
    background-color: #f94144;
}
